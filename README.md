# AI Resume Screening & Candidate Verification System

## Overview
This project is an AI-powered web application that automates resume screening by comparing candidate resumes with job descriptions and required skills.

It helps HR professionals quickly evaluate candidates by generating:
- Match Score (%)
- Predicted Job Category
- Skill Match Analysis
- Final Hiring Decision (PASS/REJECT)

---

## Features
- Resume analysis using NLP (TF-IDF + Cosine Similarity)
- Skill-based matching system
- Job category prediction using Naive Bayes
- PDF resume parsing
- Dual input (PDF upload or text paste)
- Threshold-based filtering
- Real-time results with decision output
- Deployment-ready full-stack application

---

## AI/ML Techniques Used
- TF-IDF Vectorization
- Cosine Similarity
- Naive Bayes Classification
- Custom Skill Matching Algorithm

## How It Works
1. HR enters job description and required skills
2. Candidate uploads resume or pastes text
3. System processes resume using NLP
4. Skills are matched against requirements
5. Match score is calculated
6. Final decision (PASS/REJECT) is generated
